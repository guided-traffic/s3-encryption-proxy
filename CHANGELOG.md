* Complete architectural refactor of S3 encryption proxy.

- Removed internal/s3client package entirely
- Unified all S3 operations into proxy handlers with direct AWS SDK integration
- Changed AES-CTR encryption to always store IV in metadata (no longer prepended)

but breaks internal API compatibility for any direct s3client usage.
